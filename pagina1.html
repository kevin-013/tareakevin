¿Que es una url?

Un localizador uniforme de recursos, denominado coloquialmente dirección web, es una referencia a un recurso web que especifica su ubicación en una red informática y un mecanismo para recuperarlo. Una URL es un tipo específico de Identificador uniforme de recursos, aunque muchas personas usan los dos términos indistintamente

¿Que es un dominio?

Un nombre de dominio es una cadena de identificación que define un ámbito de autonomía administrativa, autoridad o control dentro de Internet. Los nombres de dominio se utilizan en varios contextos de redes y con fines de asignación de nombres y direcciones específicos de la aplicación.

¿Que es una cookie?

Una cookie HTTP es un pequeño fragmento de datos que el navegador web almacena en la computadora del usuario mientras navega por un sitio web. Las cookies fueron diseñadas para ser un mecanismo confiable para que los sitios web recuerden información con estado o registren la actividad de navegación del usuario.
