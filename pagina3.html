¿Que es PHP?

PHP es un lenguaje de secuencias de comandos de propósito general especialmente adecuado para el desarrollo web. Fue creado originalmente por el programador danés-canadiense Rasmus Lerdorf en 1994.

¿Que es aspx?

Un archivo con la extensión de archivo ASPX es un archivo de Active Server Page Extended que está diseñado para el marco de trabajo ASP.NET de Microsoft. También se llaman formularios web.NET.
